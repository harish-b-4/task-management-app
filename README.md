# Task Management App (Vite + React + Tailwind CSS)

A simple and responsive Task Management Application built using Vite, React, and Tailwind CSS. This project allows users to easily create, view, edit, and delete their tasks in an organized table format.

# Features

Create Task: Add a new task with Title, Description, Priority, and Status.

Task Table View: All saved tasks are displayed on the home screen in a clean table layout.

Edit & Update: Modify task details anytime with a smooth and user-friendly workflow.

Delete Task: Remove tasks instantly.

Responsive UI: Fully responsive and styled with Tailwind CSS.

Fast Build: Powered by Vite for lightning-fast development.

# Tech Stack

Vite – Fast development environment

React.js – Frontend framework

Tailwind CSS – Styling & layout

# Purpose

This project demonstrates CRUD operations, React state management, page navigation, and clean UI design — ideal for beginners learning React or showcasing frontend development skills.
